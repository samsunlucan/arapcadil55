## 🌐 Veri Kaynakları

### CoinGecko API
- **URL:** https://www.coingecko.com/api
- **Lisans:** Ücretsiz katman mevcut (10-50 req/min)
- **Kullanım:** Kripto para fiyat ve market cap verisi
- **Koşullar:** Attribution gerekli, ticari kullanım için onay gerekebilir

### Binance Public API
- **URL:** https://api.binance.com
- **Lisans:** Ücretsiz (kimlik doğrulama gerektirmez)
- **Kullanım:** OHLCV verisi, emir defteri
- **Koşullar:** Rate limiting uygulanır (1200 req/min)
